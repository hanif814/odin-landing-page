# Odin Landing Page

This project is part of The Odin Project Foundations curriculum.

The goal of this project is to build a complete landing page using HTML and CSS, with a strong focus on layout, Flexbox, and visual hierarchy.

Skills demonstrated:
- Semantic HTML structure
- CSS Flexbox layout
- Typography and color systems
- Git and GitHub workflow
